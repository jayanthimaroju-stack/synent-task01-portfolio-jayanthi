# Personal Portfolio Website

## 📌 Project Overview

This project is a responsive personal portfolio website developed using HTML and CSS. It showcases my personal information, skills, education, projects, hobbies, and contact details in a clean and professional layout.

The website acts as an online portfolio that helps visitors understand my background and technical skills.

---

## 🎯 Objectives

- Build a personal portfolio website.
- Practice HTML and CSS concepts.
- Present personal and professional information in an attractive way.
- Improve front-end web development skills.

---

## 🛠️ Technologies Used

- HTML5
- CSS3

---

## ✨ Features

- Responsive and clean user interface
- About Me section
- Skills section
- Projects section
- Education details
- Hobbies section
- Contact information
- Hover effects for better user experience

1. Download or clone the repository.
